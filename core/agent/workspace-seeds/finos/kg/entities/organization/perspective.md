title: Perspective

A FINOS open-source project: an interactive visualization component for large,
real-time datasets — a fast, memory-efficient streaming query engine (C++, compiled
to WebAssembly and Python) with Apache Arrow support, plus a UI custom element and
JupyterLab widget. Originated at [[JPMC]]. Lead Maintainer: [[Andrew Stein]]. Core
maintainer: [[Tim Paine]].

Source: https://github.com/finos/perspective
