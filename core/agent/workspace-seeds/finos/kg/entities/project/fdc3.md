An open standard for financial-desktop applications to interoperate and exchange
data (257 stars). Related repos: FDC3-Sail (Electron reference implementation),
FDC3-conformance-framework, fdc3-dotnet, backplane (FDC3 Desktop Agent Bridge).
[[Rob Moffat]] is the FINOS point-of-contact / Senior Technical Architect for FDC3.
[[Kris West]] ([[NatWest Group]]) is the lead maintainer. Other contributors include
[[Brian Ingenito]] ([[Morgan Stanley]]).
