## Updating this file

All changes to the maintainer list are managed openly:

- **Submit a Pull Request** to this file for any addition, removal, or update.
- **If the project's governance requires a vote**, document or link to the vote
  outcome in the PR description or comments.
- This process creates a public audit trail of project leadership over time.

Please email **help@finos.org** whenever this file is updated with a change to
maintainership.
