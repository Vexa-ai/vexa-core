# Code of Conduct

Vexum Core follows the [FINOS Community Code of Conduct](https://www.finos.org/code-of-conduct),
which is based on the [Linux Foundation Code of Conduct](https://lfprojects.org/policies/code-of-conduct/).

We are committed to providing a friendly, safe and welcoming environment for all,
regardless of background or identity.

Please report any unacceptable behavior to [help@finos.org](mailto:help@finos.org).
