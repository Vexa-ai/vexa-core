  <Card title="Meetings as the first-class source of knowledge" icon="video" horizontal>
    [Meetings](/core/meetings) are the richest source of context for enterprise intelligence — and they
    vanish if uncaptured. Vexa captures transcripts from [Google Meet](/core/meetings),
    [Zoom](/core/meetings), and [Microsoft Teams](/core/meetings) in real time, natively.
  </Card>
  <Card title="Knowledge as code" icon="code" horizontal>
    AI totally disrupted software development; knowledge work is next. To an LLM, any text is a token.
    Vexa parses your meetings and emails into a living, Markdown-based
    [workspace](/concepts#workspace). [Agents](/core/agents) treat this continuous wiki as their working
    directory — reading from and writing to it natively, just like a developer navigating a codebase.
  </Card>
  <Card title="A CLI agent is all you need — sandboxed and scalable" icon="terminal" horizontal>
    A CLI agent on a Linux machine is all you need to build software. Vexa scales that execution model to
    enterprise infrastructure — safely — by sandboxing [agents](/core/agents) into
    [containers](/concepts#container).
  </Card>
</CardGroup>
