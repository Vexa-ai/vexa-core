# calm/patterns — reusable CALM patterns

CALM patterns a concrete architecture must conform to (`calm validate -p`). One pattern per file.

| File | Pattern |
|---|---|
| `meeting-intelligence.pattern.json` | a self-hostable meeting-intelligence runtime — single authenticated edge, capture worker, collector, copilot, render-only client, and a declared STT egress boundary. Fail-closed. |
